# Endless Soju Party!

Endless Soju Party!はESP32-C3-MINI-1-N4を使用したBluetooth-シリアル変換基板です。

## 仕様

|        機能         | 個数 | 備考                     |
| :-----------------: | ---: | :----------------------- |
|         MCU         |    1 | ESP32-C3-MINI-1-N4       |
|         CAN         |    1 | 電源ピンは5V、CAN FD対応 |
|        UART         |    1 | 電源ピンは5V             |
|         USB         |    1 | 書き込み用               |
|  リセットスイッチ   |    1 |                          |
| 5V-3.3Vレギュレータ |    1 | 外部電源からの供給用     |
|     寸法(外形)      |      | 30mm x 57.5mm            |
|   寸法(USB部以外)   |      | 30mm x 45mm              |
|   寸法(固定穴間)    |      | 24.4mm                   |

## 名前の由来

**E**ndless **S**oju **P**artyです。シャレです。ただの。
